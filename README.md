# DevCamper project API

> Backend for DevCamper application to be developed in React.

## Usage

1. Copy env variables into config/config.env and rename the file to .env once deployed.

## Install Dependencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in prod
npm start
```
